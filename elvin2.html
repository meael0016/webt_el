Etiquetas básicas de html

Aquí aprenderás las etiquetas básicas de html así entenderás más el programa Y también aprenderas lo basico del html para tus pequeños proyectos

Etiquetas H

Las etiquetas H se pueden usar como títulos y subtítulos de un parrafo.estás están desde la h1 hasta la h6, la h1 es la más grande de todas, mientras que desde h2 hasta h6 tiene un tamaño más pequeño.

Pero ten cuidado con el uso de la etiqueta h1 Google te puede penalizar tu página al usar demasiado esta etiqueta

Ejemplo

Mi página web se llama webt_El y por este medio el Buscador de Google me ayara mas rápido por eso aqui utilice una etiqueta h1.con esto el Buscador de Google me ubica más rapido en la posición,te recomiendo que sólo lo utilices una vez así no tendrás problemas. con el h2 hasta el h6 no pasa esto lo puedes usar cuantas veces quieras.
