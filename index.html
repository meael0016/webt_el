Proposito del programa

El programa "Webt_El" fue creado con el objetivo de elaborar páginas web sencillas que ayuden al usuario a aprender las etiquetas del html más rápido. Gracias al uso de la voz puedes poner textos en algunas etiquetas principales de html. Este programa va dedicado a los que empiezan a aprender html, es muy sencillo para lo que quiero lograr, esto es lo que puedo hasta ahora. Su próxima actualización incluirá más dominios para controlar las etiquetas, lo que tengo en mente con este proyecto es muy ambicioso pero esto es lo que puedo dar hasta ahora, espero que les sirva de mucho.

Todo el contenido con etiquetas p,h y li estan hechos con este programa ganando tiempo con los textos de la pagina.

Audio

El boton audio es el encargado de reconocer la voz al principio aparece desactivado solo se activa cuando las etiquetas que permiten vos sean pulsadas.

H,P,li

Son las unicas que pueden contener el contenido del audio

Abre html y cierra html

Antes de comenzar a escribir un documento html tienes que abrir estas etiquetas el html la cabeza y el cuerpo.

Despues que hayas escrito todo el documento tienes que cerrar el /body,/html.

Tipos de listas ul y ol

Hay dos tipos de listas en html la (ol) numerica y la (u1) es con viñetas antes de pulsar li tienes que escoger el tipo de lista, despues que escoge el tipo de lista y añadirle audio a la lista tienes que cerrar el tipo de lista esto lo deje al mando del usuario asi aprendera mas.

Borrar contenido

Contenido del archivo

Se utiliza para ver todo el contenido del archivo y lo que acabas de poner en el.

Agregar contenido al archivo

Se utiliza para cambiar el contenido que hayas cambiado en el display lo que allas puesto en el display se actualizara el archivo y lo que alla dentro del display se guardara.

Crear archivo

Se utiliza para crear un archivo TXT o HTML.

Elegir archivo

Se utiliza para elegir el archivo que hayas creado o que ya tengas el escritor o en cualquier parte de la peseta.

Etiqueta pulsada lista para vos

En este pequeño display se pondrá la etiqueta que está lista para vos, para que el usuario sepa cual esta utlizando.
